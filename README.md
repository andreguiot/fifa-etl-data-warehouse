Este projeto tem como objetivo normalizar uma base de dados bruta de sobre da fifa aceitamos a sugestão do professor , garantindo que nenhuma informação seja perdida e, posteriormente, estruturar um Data Warehouse (DW) com um processo completo de ETL (Extract, Transform, Load).
Ele demonstrar a criação de um ambiente PostgreSQL, desenvolvido a pedido do nosso professor  para o curso de Banco de Dados II do CEFET-RJ.
